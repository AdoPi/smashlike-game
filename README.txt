le projet se lance de deux manieres :

un menu graphique dans le package menu (classe Main)

le jeu sans menu qui se lance depuis le Main dans le default package

Nous avons utilisé le Main du package par défaut pour effectuer nos tests.


Enfin, vous pouvez compiler l'intégralité du projet à l'aide de la commande 'ant'
grâce au build.xml.

Le résultat de la compilation sera dans build/classes/
il suffira de lancer Main ou menu.Main pour démarrer le projet.
